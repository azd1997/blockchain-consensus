## Go环境

- 下载的Go安装程序安装目录即为GOROOT
- GOPATH不太重要，你可以不管，但是把GOPATH/bin和GOROOT/bin添加到PATH变量里
- GOPROXY = https://goproxy.cn,direct （go env -w GOPROXY=https://goproxy.cn,direct）
- IDE就用Goland

## Go mod

包管理工具

- go mod download 或者是 go mod tidy 或者 Goland自动提示的Sync Dependencies
- 新建一个项目时，通常要 go mod init module_name。 比如 github.com/azd1997/project_name
- go mod 文件中replace等语法自行搜索

## 一个Go文件或项目的编译与运行

- go build project_dir （只编译）或 go run ... （编译后运行）


## Go基础数据类型

- 整数： intxx（定长）、int（变长） ；相应的，uint
- 浮点数： float32/float64
- 字符：byte（定长）、rune（变长，unicode字符，1~4个字节不等）
- 字符串：string
- 布尔：bool

## Go独有的数据类型以及复合类型

- rune 变长字符
- interface 接口，有点像抽象类。一个接口由若干个方法声明组成，约定了某一类事物的共同行为。
- struct 
- interface{} 空接口。 any
- slice 变长数组
- map 
- chan 

方法与函数的区别：
- 方法指一个类独有的（只有这个类的对象才能调用的）函数，并且这个函数默认有一个接收者，就是该类实例化的对象，该对象自动作为函数/方法的参数传入。
